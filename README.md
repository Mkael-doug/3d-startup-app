O **ROBOY** é uma landing page moderna e interativa desenvolvida com o intuito de praticar e dominar a integração de elementos 3D no desenvolvimento web. Utilizando **React**, **Vite** e **Tailwind CSS**, o projeto conta com uma experiência rica e imersiva através do uso de modelos 3D criados no **Spline** e animações dinâmicas ativadas ao rolar a página.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e bibliotecas:

- **[React](https://react.dev/) (v19)**: Biblioteca JavaScript para construção de interfaces de usuário reativas.
- **[Vite](https://vite.dev/)**: Ferramenta de build de última geração para desenvolvimento web extremamente rápido.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para estilização rápida, moderna e responsiva.
- **[@splinetool/react-spline](https://spline.design/)**: Integração direta e interativa de cenas e modelos 3D criados no Spline.
- **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)**: Biblioteca para aplicação de animações fluidas baseadas na rolagem da página.
- **[Boxicons](https://boxicons.com/)**: Biblioteca de ícones vetoriais modernos e leves para estilização.

---

## ✨ Funcionalidades Principais

- **Integração 3D Interativa**: Um robô 3D estilizado e interativo renderizado em tempo real na tela inicial, interagindo com o cursor e cliques.
- **Design Totalmente Responsivo**: Layout e navegação otimizados para dispositivos móveis, tablets e desktops.
- **Animações na Página**: Efeitos de surgimento e transição de elementos ao longo da navegação, alimentados pela biblioteca AOS.
- **Menu Mobile Integrado**: Menu de navegação responsivo no estilo hamburguer para telas menores.
- **Estética Premium**: Efeitos de brilho traseiros (glow effects) e gradientes harmonizados para simular interfaces de startups de tecnologia modernas.

## 🚀 Como Executar o Projeto Localmente

Siga os passos abaixo para clonar e rodar o projeto em sua máquina:

### Pré-requisitos
Certifique-se de ter instalado em sua máquina:
* [Node.js](https://nodejs.org/) (recomendado versão LTS)
* Um gerenciador de pacotes como o `npm` (instalado automaticamente junto com o Node.js)

### Instruções

1. **Clonar o Repositório:**
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```

2. **Acessar a Pasta do Projeto:**
   ```bash
   cd seu-repositorio
   ```

3. **Instalar todas as Dependências:**
   ```bash
   npm install
   ```

4. **Executar o Servidor de Desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Visualizar no Navegador:**
   Abra o navegador e acesse o endereço fornecido no terminal (geralmente [http://localhost:5173](http://localhost:5173)).
